# 🚀 smash-inc.co.jp 公開クイックスタートガイド

**所要時間**: 約30分〜1時間

---

## ステップ1: メインサイトをアップロード（15分）

### Xserverにログイン
👉 https://www.xserver.ne.jp/login_server.php

### ファイルマネージャーを開く
1. サーバーパネルから「ファイルマネージャー」をクリック
2. `/smash-inc.co.jp/public_html/` に移動

### 以下のファイルをアップロード
```
✅ index.html
✅ news-detail.html
✅ style.css
✅ assets/ フォルダ（中身すべて）
```

### 確認
👉 https://smash-inc.co.jp にアクセス
→ トップページが表示されればOK！

---

## ステップ2: WordPress（News用）を設置（10分）

### サブドメインを追加
1. サーバーパネル →「サブドメイン設定」
2. サブドメイン名: `news`
3. 対象ドメイン: `smash-inc.co.jp`
4. 追加

### WordPressをインストール
1. サーバーパネル →「WordPress簡単インストール」
2. ドメイン選択: `news.smash-inc.co.jp`
3. 以下を入力:
   - ブログ名: `SMASH News`
   - ユーザー名: （お好きなもの）
   - パスワード: （強力なもの）
   - メールアドレス: （管理用）
4. インストール実行

### 管理画面にログイン
👉 https://news.smash-inc.co.jp/wp-admin/
→ インストール時に設定したユーザー名・パスワードでログイン

---

## ステップ3: WordPressの初期設定（5分）

### パーマリンク設定
1. WordPress管理画面：「設定」→「パーマリンク」
2. 「投稿名」を選択
3. 保存

### 検索エンジンブロック（推奨）
1. 「設定」→「表示設定」
2. 「検索エンジンがサイトをインデックスしないようにする」にチェック
3. 保存

---

## ステップ4: テスト投稿（5分）

### 記事を投稿
1. WordPress管理画面：「投稿」→「新規追加」
2. タイトル: `テスト投稿`
3. 本文: 適当に数行入力
4. アイキャッチ画像を設定（推奨）
5. 「公開」をクリック

### API動作確認
👉 https://news.smash-inc.co.jp/wp-json/wp/v2/posts
→ JSON データが表示されればOK

---

## ステップ5: 連携確認（5分）

### メインサイトでNews表示を確認
👉 https://smash-inc.co.jp
→ Newsセクションにスクロール
→ 投稿した記事が表示されていればOK！

### 詳細ページを確認
1. 記事カードをクリック
2. 詳細ページが表示されることを確認

---

## 🎉 完了！

おめでとうございます！サイトが公開されました。

---

## ⚠️ うまく表示されない場合

### Newsが表示されない → CORS設定

WordPress側に `.htaccess` を追加:

**FTPまたはファイルマネージャーで以下のファイルを編集:**
```
/news.smash-inc.co.jp/public_html/.htaccess
```

**一番上に以下を追記:**
```apache
<IfModule mod_headers.c>
    <If "%{REQUEST_URI} =~ m#^/wp-json/#">
        Header set Access-Control-Allow-Origin "https://smash-inc.co.jp"
    </If>
</IfModule>
```

保存して、ブラウザをリロード（Ctrl+F5）

## 📝 日常運用

### 記事を投稿するには
1. https://news.smash-inc.co.jp/wp-admin/ にログイン
2. 「投稿」→「新規追加」
3. タイトル・本文・アイキャッチを設定
4. 「公開」

→ 数秒後、自動的に https://smash-inc.co.jp のNewsに反映！

### サイトの内容を更新するには
1. ローカルで `index.html` や `style.css` を編集
2. FTPまたはファイルマネージャーで該当ファイルを上書きアップロード
3. ブラウザでキャッシュクリア（Ctrl+F5）して確認
